Edit or delete an SSO connection

Abstract

Describes how to edit and delete an SSO connection in the Sitecore Cloud Portal.

You can edit the settings of your existing SSO connections or delete an SSO connection you don't need.

Edit an SSO connection

If your identity provider settings change or you entered the wrong details, you can edit the settings of an existing SSO connection. If you edit a setting, we recommend that you test your SSO connection again.

To edit an existing SSO connection:

  1. Navigate to the Sitecore Cloud Portal SSO page and click Edit for the connection you want to edit.

  2. In the Edit SSO connection dialog, update the relevant fields, then click Save.

Delete an SSO connection

When you delete an SSO connection, team members that belong to the deleted SSO connection can still log in to the Sitecore Cloud Portal using the default Sitecore authentication. However, if team members don't remember their password, they must first reset it.

To delete an SSO connection:

  1. Navigate to the Sitecore Cloud Portal SSO page and click Delete for the connection you want to delete.

  2. In the Delete active SSO connection? dialog, enter the name of the connection you want to delete, then click Delete.
